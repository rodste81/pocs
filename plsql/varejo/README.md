# Projeto Varejo (PL/SQL)

Este projeto tem como objetivo gerenciar rotinas de varejo utilizando PL/SQL.

## Estrutura de Pastas

- **tables/**: Scripts de criação de tabelas (DDL).
- **packages/**: Packages (Spec e Body) com a lógica de negócio.
- **views/**: Views para relatórios e consultas.
- **scripts/**: Scripts de carga de dados ou execução de rotinas.

## Como Iniciar

1. Execute os scripts em `tables/` para criar o schema.
2. Compile os packages em `packages/`.
